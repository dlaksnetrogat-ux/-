Инструкция по запуску проекта "Браузер" для начинающих.

Чтобы запустить этот браузер на компьютере, выполните следующие простые шаги:

1. Установите Node.js
   Если у вас еще не установлена программа Node.js, скачайте ее с официального сайта: https://nodejs.org/ (рекомендуется версия LTS для большинства пользователей) и установите.

2. Установите зависимости проекта
   - Откройте папку с проектом.
   - Щелкаем левой кнопкой мыши по адресной строке проводника (самая верхняя строка, где написан текущий путь до папки).
   - Стираем путь, пишем "cmd" (без кавычек) и нажимаем Enter. Появится черное окно командной строки.
   - В командной строке напишите команду:
     npm install
   - Нажмите Enter и подождите. Программа скачает и установит необходимые компоненты для работы браузера (появится папка `node_modules`).

3. Запустите проект
   - В том же самом черном окне командной строки напишите команду:
     npm start
   - Нажмите Enter.

После этого запустится ваш браузер!
